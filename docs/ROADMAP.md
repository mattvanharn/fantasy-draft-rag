# Project Roadmap

## Phase 1: RAG Pipeline MVP

1. **Project Setup** — venv, git, Ollama, verify dependencies
2. **Data Collection** — Sample player stats (nfl_data_py), expert rankings
3. **Document Processing** — Loaders, chunking, text conversion
4. **Embeddings & Vector Store** — ChromaDB, retrieval
5. **RAG Pipeline** — LangChain chain, Ollama LLM, citations
6. **Evaluation & Polish** — Testing, tuning, documentation

## Phase 2: Web UI

Streamlit or Gradio chat interface.

## Phase 3: Draft Recommendations

Roster-aware suggestions, PyTorch prediction models.

## Phase 4+: Advanced Features

Live draft tracker, trade analysis, league settings.
